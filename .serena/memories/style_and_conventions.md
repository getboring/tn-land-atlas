# Style and conventions
Follow `AGENTS.md` and `CLAUDE.md`. ESLint, not Biome. TypeScript strict, zero `any` in `src/` or `functions/`. Supabase runtime access is server-only in `functions/api/property.ts`; never add `VITE_SUPABASE_*`. All Pages Functions accepting input must validate via `_validate.ts` and return 400 on null. ArcGIS query escaping is defense-in-depth, not a replacement for validation. Map container must use inline absolute positioning and ResizeObserver. Long-lived MapLibre handlers must call current refs, not stale closures. Parcels load only at zoom >= 13. Search renders max 200 rows. Money from ArcGIS is whole dollars for `fmtMoney`.
